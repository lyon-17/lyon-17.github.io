Ejercicio 2

Modificar el ejercicio anterior del objeto Factura para crear una pseudoclase llamada Factura y que permita crear objetos de ese
tipo. Se deben utilizar las funciones constructoras y la propiedad prototype.
Para instanciar la clase, se debe utilizar la instrucción Factura(cliente, elementos), donde cliente también es una pseudoclase que
guarda los datos del cliente y elementos es un array simple que contiene las pseudoclases de todos los elementos que forman la
factura.
