formas de crear un array

var numeros = new Array();
var numeros = new Array(10);
var numeros = [];
var numeros = [10];

Tipos de arrays

indexados : var numeros = [1,2,3,4,5]; //cada valor tiene un indice numerico
mixto: var numeros = [ 'nombre' : 'Alejandro' , 'numero' : 1 , 'booleando' : 'false']; //indice : valor, indice2, valor2...

Formas de eliminar elementos

delete numeros[2]; //elimina el elemento en esa posicion y deja un hueco.
numeros.splice(2); //deja el tamaño del array a dicho valor.
numeros.pop(); //elimina el ultimo elemento y lo devuelve.
numeros.shift(); //elimina el primer elemento y lo devuelve.

Formas de introducir datos

numeros[0] = 1;
numeros = new Array (1,2,3,4,5,6,7);
numeros = [1,2,3,4,5,6,7];
numeros.unshift(3) // añade elemento al comienzo de array
numeros.push(3);

Formas de recorrer un array

Bucle for

for (i=0;i < numeros.length ; i++)
contenido.innerHTML(numeros[i] + ' ,');

for each

for each (var clave in numeros)
contenido.innerHTML(clave + ' br/');

Otros metodos

concat(); //une dos o mas arrays y devuelve la copia.
join(); //une todos los elementos del array en una cadena
reverse(); //Invierte el orden del array
sort(); //ordena el array por orden alfabetico
toString(); //convierte el array en una cadena
slice(index,index2); //Crea un array. Comenzara desde el index incluido hasta index2 no incluido, o el final si no se le da ningun valor al index2 y creara el array con esos elementos. Si el indice es negativo comenzara por el final del array.
